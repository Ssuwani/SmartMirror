# SmartMirror

## Purpose
[Magic Mirror]라는 오픈소스 프로젝트를 통해서 스마트미러를 많이 만들었었다. 학생들한테 강의도 하고 포스팅도 했었지만 정작 어떻게 동작하는지 제대로 알지 못했다. 내가 잘 아는 파이썬 코드를 통해서 SmartMirror를 만들어보자!

## Goal

### Design
디자인적으로 목표는 다음과 같은 이미지 중에 하나가 되지 않을까 싶다.. 예뻐보이는 스마트미러...
